# 🕵️ WhiteSINT

> **Ethical, legal, and open-source OSINT multitool — built with transparency in mind.**

WhiteSINT is a Python-based OSINT toolkit designed for cybersecurity researchers, ethical hackers, and curious minds who believe that intelligence gathering must always stay legal, respectful, and responsible.

## ⚠️ Disclaimer

WhiteSINT is intended **strictly for educational and ethical use only**.
Always ensure you have proper authorization before running any tool against a target.
The authors are not responsible for any misuse of this software.

---

## 🛠️ Tools
- IP Lookup
- WHOIS Lookup
- Phone Lookup
- Ports Scanner
- Identity Generator
- Password Generator
- Multiple Searcher
- Roblox Lookup (ID)
- Discord Webhook Lookup
- Discord Webhook Sender
- Discord Webhook Remover

---

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/no0xpy/WhiteSINT.git
cd WhiteSINT
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run WhiteSINT

```bash
python menu.py
```

---

## 🖥️ Requirements

- Python 3.13+
- pip
- Internet connection

> *"Information is not a weapon — it's a responsibility."*

WhiteSINT was built on three pillars: **legality**, **transparency**, and **ethics**.
Every tool included is designed to be used only on targets you own or have explicit permission to investigate.
