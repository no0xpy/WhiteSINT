# 🕵️ WhiteSINT

> **Ethical, legal, and open-source OSINT multitool — built with transparency in mind.**

WhiteSINT is a Python-based OSINT toolkit designed for cybersecurity researchers, ethical hackers, and curious minds who believe that intelligence gathering must always stay legal, respectful, and responsible.
<img src="https://github.com/no0xpy/WhiteSINT/blob/main/screen.png">
---

## ⚠️ Disclaimer

WhiteSINT is intended **strictly for educational and ethical use only**.
Always ensure you have proper authorization before running any tool against a target.
The authors are not responsible for any misuse of this software.

---

## 🛠️ Tools

| # | Tool | Description |
|---|------|-------------|
| 1 | **IP Lookup** | Geolocate an IP address, retrieve ISP, country, city and more |
| 2 | **WHOIS Lookup** | Get domain registration info — owner, creation date, registrar and more |
| 3 | **Phone Lookup** | Retrieve informations about a phone number |
| 4 | **Ports Scanner** | Scan open ports on a target host to map its attack surface |
| 5 | **Identity Generator** | Generate realistic identities |
| 6 | **Password Generator** | Create strong random passwords |
| 7 | **Multiple Searcher** | Search on DuckDuckGo, Google, Brave, Bing, Yandex quickly |

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/no0xpy/WhiteSINT.git
cd WhiteSINT
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run WhiteSINT

```bash
python menu.py
```

---

## 🖥️ Requirements

- Python 3.13+
- pip
- Internet connection (for lookup tools)

## 👁️ Philosophy

> *"Information is not a weapon — it's a responsibility."*

WhiteSINT was built on three pillars: **legality**, **transparency**, and **ethics**.
Every tool included is designed to be used only on targets you own or have explicit permission to investigate.
